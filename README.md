# Python Hadoop租房分析系统

## 核心功能
- 用户登录注册
- 租房数据可视化大屏
- 房屋信息推荐
- 租房数据查询
- 房屋资讯浏览
- 用户管理
- 房屋信息管理
- 租房数据管理
- 系统管理

## 适用场景
- 房地产市场分析
- 租房需求预测
- 房屋租赁策略制定
- 城市规划与住房政策制定

## 技术特点
- 使用Python语言开发，结合Hadoop进行大数据处理
- 基于 Django 框架搭建，采用 MySQL 数据库存储数据
- 采用 Vue.js 实现前端界面，提供良好的用户体验
